📊 Understanding Report Insights on LaporFix

The Report Insights feature gives you a visual overview of issue reporting trends, resolution times, and community engagement. This guide explains what data you can view and how to use the insights dashboard.

═══════════════════════════════════════════════════════════════════
ACCESSING INSIGHTS
═══════════════════════════════════════════════════════════════════

From the Issue tab (Status Tracker), tap the **Insights icon** (📊) in the top-right corner. This opens the insights dashboard.

═══════════════════════════════════════════════════════════════════
FILTER CONTROLS
═══════════════════════════════════════════════════════════════════

At the top of the insights screen, you'll see two filter controls:

┌────────────────────────────────────────────────────────────────┐
│ CATEGORY FILTER                                                │
│                                                                │
│ Choose what you want to analyze:                              │
│   • Overview — Shows data for all issue categories combined   │
│   • Specific category — Shows data for just that category     │
│     (e.g., Pothole, Water Leakage, Garbage Overflow)          │
│                                                                │
│ Available categories are pulled from your submitted reports.  │
└────────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────────┐
│ TIME PERIOD FILTER                                             │
│                                                                │
│ Select the time range for analysis:                           │
│   • Last 24 Hours — Hourly breakdown (4 buckets: 12am-6am,    │
│     6am-12pm, 12pm-6pm, 6pm-12am)                             │
│   • Last 7 Days — Daily breakdown                             │
│   • Last 30 Days — Weekly breakdown                           │
│   • Last 90 Days — Monthly breakdown                          │
│                                                                │
│ The dashboard updates instantly when you change the period.   │
└────────────────────────────────────────────────────────────────┘

HOW DATA IS CALCULATED
═══════════════════════════════════════════════════════════════════

  RESOLUTION TIME
    Measured from the time the issue was submitted to when it was
    marked as "Completed". Issues still in progress or submitted are
    not included in resolution time calculations.

  TIME PERIOD FILTERING
    Issues are included if their submission date falls within the
    selected period. For "Last 24 Hours", the period runs from
    midnight to midnight.

  TREND COMPARISON
    When viewing a specific category, trend percentages compare the
    current period to the immediately previous period of the same
    length:
      • Last 7 Days → compares to the 7 days before that
      • Last 30 Days → compares to the 30 days before that
      • Last 90 Days → compares to the 90 days before that

  ENGAGEMENT RANKING
    Issues are ranked by (likes + comments). If two issues have the
    same total, the one with more comments ranks higher. If still tied,
    the one with more likes ranks higher. If still tied, the most
    recently updated issue ranks higher.

═══════════════════════════════════════════════════════════════════
LIVE UPDATES
═══════════════════════════════════════════════════════════════════

The insights dashboard updates automatically in real time when:
  • A new issue is submitted
  • An issue status changes (e.g., moves from "In Progress" to "Completed")
  • An issue receives new likes or comments

You don't need to manually refresh the page.

═══════════════════════════════════════════════════════════════════
💡 TIPS
═══════════════════════════════════════════════════════════════════

  • Use Overview mode to identify which categories need the most attention
  • Switch to a specific category to see location hotspots for that issue type
  • The "Most Active Area" card helps you see where problems are concentrated
  • Top Engagements shows which issues the community cares about most
  • Trend indicators help you see if resolution times are improving or worsening
  • The "Unresolved" count includes both submitted and in-progress issues
  • Deleted issues are excluded from all calculations
